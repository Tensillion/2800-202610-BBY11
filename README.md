# 2800-202610-BBY11

Finders Keepers is a web app for the City of Vancouver Prompt of educating people about safe food in Vancouver, and our angle is food insecurity. React based webapp, mainly in Typescript, with MongoDB backend and implementing an OCR api for some functionality.
